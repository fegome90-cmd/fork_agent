# Project: Fork Agent

## General Instructions

- You are an expert in terminal orchestration and AI-powered development.
- Your primary goal is to assist users in managing complex, concurrent tasks using various AI agents.
- When interacting with the user, be proactive and suggest commands and workflows that leverage the full capabilities of Fork Agent.

## Project Overview

Fork Agent is a powerful command-line tool that enhances terminal workflows through several key features:

- **Terminal Orchestration:** Manages multiple terminal sessions to handle complex and concurrent tasks.
- **Advanced Multi-Agent Support:**
    - **Raw CLI:** For direct shell command execution.
    - **Claude Code:** For AI-assisted programming interactions.
    - **Codex CLI:** For code generation and execution assistance.
    - **Gemini CLI:** For intelligent commands powered by the Gemini model.
- **Conversational Context:** Utilizes chat history to enrich agent prompts, enabling smarter execution and summarization.
- **Robust Multi-Platform Support:** Works seamlessly across macOS, Windows, and Linux.

## Superpowers & Extensibility

Fork Agent is designed to be highly extensible, allowing you to enhance its capabilities with custom logic and automated processes.

- **Skills:** Package on-demand expertise into discoverable capabilities. Skills provide specialized instructions and resources that can be activated by the agent when needed to perform specific tasks, such as security auditing or cloud deployments. They are discovered from the workspace, user, and extensions.

- **Workflows:** Define and automate complex multi-step tasks by creating sequences of commands and agent interactions. Workflows ensure that intricate processes are executed consistently and reliably, saving you time and reducing errors.

- **Custom Commands:** Create personalized shortcuts for frequently used prompts and operations. Custom commands allow you to tailor Fork Agent to your specific needs and streamline your personal workflow, making you more efficient.

## Coding Style

- Follow standard shell scripting best practices.
- For AI-related interactions, prioritize clear and concise language.
- When generating code, adhere to the dominant style of the target language.
