# Prompt: Resumen de cambios para usuario

Objetivo: generar un resumen conciso de cambios propuestos o realizados en el repositorio, orientado a revisión humana.

Plantilla:

```
Eres un asistente que resume cambios en un repositorio de código.
Entrada: lista de archivos modificados, breves notas de cada cambio, riesgos conocidos.
Salida: un resumen en 3-6 líneas indicando qué se cambió, por qué, y la acción recomendada (revisar, probar, merge).
```

Ejemplo de uso y etiquetas: `break`, `performance`, `security`.
